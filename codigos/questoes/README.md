# questoes

Projetos resolvidos da `PROVA_SIMULADA.md`, um por questao, prontos para abrir no IntelliJ e clicar em `Run`.

Projetos criados:

- `questao01-laboratorio-impressao`
- `questao02-deposito`
- `questao03-sistema-de-logs`
- `questao04-banco`
- `questao05-arquivo-compartilhado`
- `questao06-largada-de-corrida`
- `questao07-servidor-com-limite`
- `questao08-ponte-estreita`
- `questao09-processamento-paralelo`
- `questao10-fila-de-atendimento`
- `questao11-atendimento-com-prioridade`
- `questao12-ordem-entre-threads`

Fluxo:

1. Abra a pasta da questao desejada no IntelliJ.
2. Aguarde importar o `pom.xml`.
3. Edite apenas `src/main/java/Main.java` se quiser adaptar.
4. Clique em `Run` na configuracao `Main`.
